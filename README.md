# uefa-euro-2020-data-analysis

In this project, I worked with UEFA Euro 2020 match event data and used PySpark to process and analyze it.

The dataset contains UEFA Euro 2020 match event data, including:

- Match information  
- Teams and players  
- Event types (passes, shots, fouls, etc.)

What This Project Does:

- Loads match event data from CSV/S3 into PySpark
- Converts and structures the data into Spark DataFrames
- Cleans and standardizes columns (data types, formats)
- Performs transformations to make the data analysis-ready
- Runs basic aggregations and insights on match events
